# Tienda 3D – Filamentos  
Autor: Federico Gastón Villoria  

Este proyecto es una página web desarrollada como Trabajo Práctico para la materia de Diseño y Desarrollo Web.  
La finalidad del sitio es simular una tienda de filamentos para impresoras 3D, incluyendo navegación interna, productos, reseñas y un formulario funcional de contacto.

## Contenido del Proyecto
- Estructura semántica en HTML utilizando: header, nav, main, section y footer.
- Formulario de contacto funcional implementado con Formspree.
- Archivo CSS externo (styles.css) con estilos generales, Google Fonts, backgrounds, sombras y animaciones.
- Diseño responsivo mediante Flexbox (Productos), Grid (Reseñas) y Media Queries.
- Contenido multimedia con imágenes de los productos.
- Navegación interna con enlaces a Inicio, Productos, Reseñas y Contacto.
- Proyecto subido y accesible mediante GitHub Pages.

## Estructura del Proyecto
index.html  
styles.css  
README.md  
img/filamentos/ (contiene todas las imágenes utilizadas en el proyecto)

## Objetivo del Trabajo
Aplicar los conceptos aprendidos de HTML, CSS, Flexbox, Grid y diseño responsivo para construir una página web estética, funcional y correctamente estructurada.
